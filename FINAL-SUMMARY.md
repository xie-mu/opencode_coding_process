## 📚 文档内容精华

### 1. 核心概念总结
- **OpenClaw 是什么**：WhatsApp + Telegram + Discord + iMessage 网关
- **代理系统**：基于 pi-mono 的嵌入式代理运行时
- **工作空间**：代理的"记忆"和操作指南
- **多代理支持**：多个隔离的代理系统

### 2. 安全机制
- **访问控制**：`channels.whatsapp.allowFrom` 限制
- **工具权限**：每个代理独立的工具 allow/deny 列表
- **沙箱隔离**：`sandbox.mode` 控制执行环境
- **会话管理**：独立的会话存储和认证配置

### 3. 自动化系统
- **心跳机制**：周期性检查，默认30分钟，支持智能抑制
- **定时任务**：精确的 cron 调度，支持隔离会话
- **决策流程**：根据需求选择心跳或定时任务

### 4. 代理个性化
- **SOUL.md**：定义代理人格、边界、语调
- **IDENTITY.md**：设置代理名称、类型、特征、表情符号
- **AGENTS.md**：操作指南和记忆系统

## 🛠️ 实用功能

### 快速开始
```bash
# 安装 OpenClaw
openclaw setup

# 启动网关
openclaw gateway --port 18789

# 创建定时提醒
openclaw cron add \
  --name "提醒喝水" \
  --at "5m" \
  --session isolated \
  --message "💧 该喝水啦！" \
  --delete-after-run
```

### 代理配置
```json5
{
  "agents": {
    "defaults": {
      "workspace": "~/.openclaw/workspace",
      "model": "anthropic/claude-opus-4-6"
    }
  },
  "channels": {
    "whatsapp": {
      "allowFrom": ["+15555550123"]
    }
  }
}
```
