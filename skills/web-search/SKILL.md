# 联网搜索技能

这是一个强大的网络搜索和信息查询工具，可以快速获取互联网上的最新信息和数据。

## 功能特性

- 🔍 **智能搜索** - 基于关键词的网络搜索
- 🌐 **多源查询** - 支持多个搜索引擎
- 📰 **实时信息** - 获取最新的新闻资讯
- 🔬 **技术查询** - 查找技术文档和教程
- 📊 **数据检索** - 搜索统计数据和分析

## 使用方法

```bash
# 基本搜索
web-search query --q "GPT-5.3 最新使用方法"

# 技术文档搜索
web-search tech --q "GPT-5.3 API 文档"

# 新闻资讯搜索
web-search news --q "GPT-5.3 发布新闻"

# 多关键词搜索
web-search multi --q "GPT-5.3 使用方法 2026"
```
