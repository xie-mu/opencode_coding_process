# 天气查询技能

这是一个实时天气查询和预报服务，提供准确的天气信息和预测。

## 功能特性

- 🌤️ **实时天气** - 当前天气状况和温度
- 📅 **天气预报** - 未来7天天气预报
- 🌡️ **温度转换** - 摄氏度和华氏度转换
- 💧 **湿度信息** - 湿度和降水概率
- 💨 **风速风向** - 风速和风向信息
- 🌪️ **极端天气** - 风暴、雪暴等预警

## 使用方法

```bash
# 查询当前天气
weather query --city 北京

# 获取天气预报
weather forecast --city 上海 --days 3

# 温度转换
weather convert --temp 25 --from celsius --to fahrenheit
```
