# 智能计算器技能

这是一个功能强大的高级数学计算和科学计算工具，支持多种计算模式和算法。

## 功能特性

- 🧮 **基础计算** - 加减乘除、幂运算、开方
- 🔬 **科学计算** - 三角函数、对数、指数、复数
- 📊 **统计分析** - 平均值、标准差、回归分析
- 🔢 **进制转换** - 二进制、八进制、十六进制
- 📐 **几何计算** - 面积、体积、角度计算
- 📈 **金融计算** - 复利、现值、未来值计算

## 使用方法

```bash
# 基础计算
calculator basic --expr "2+3*4"

# 科学计算
calculator scientific --func "sin(30)" --unit "degrees"

# 统计分析
calculator stats --data "1,2,3,4,5"
```
